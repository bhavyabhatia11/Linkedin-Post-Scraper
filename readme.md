## Introduction

The LinkedIn Post Scraper is a simple Chrome extension that allows users to scrape posts from their LinkedIn feed. This tool can be used for various purposes such as data analysis, research, or simply to keep track of your own posts.

The extension uses auto scroll behaviour to fetch posts without triggering linkedin's bot check. Once fetched, the extension extracts the relevant information such as the post text, number of likes, and number of comments.

## Installation

To install the LinkedIn Post Scraper extension, follow these steps:

1. Clone or download this repository.

2. Open Google Chrome and navigate to `chrome://extensions/`.

3. Enable Developer mode by clicking the toggle switch in the upper right-hand corner.

4. Click the Load unpacked button and select the downloaded folder.

## Usage

To use the LinkedIn Post Scraper extension, follow these steps:

1. Navigate to your LinkedIn feed or search for specific posts. Make sure you have `Posts` tab selected.

2. Click the extension icon in the top right-hand corner of your browser.

3. Enter the number of posts you would like to scan in the popup window.

4. Wait for the extension to finish scraping the posts. The results will be downloaded in csv file.
